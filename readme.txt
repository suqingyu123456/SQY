一：SQY简介

    SQY是我个人平时开发用的linux下C++基础库，由我本人开发，维护。主要目的是封装C++各种底层函数和系统调用，提供各种常见方法接口，并将其封装成类以供开发使用。
    因为我比较不会起名字，该库暂时以我名字缩写命名，即苏青羽——SQY。


二、SQY内容

    库中每个类都提供了对应的功能，具体功能和类的使用可以查看对应文件的内容。
    目前库已经实现的功能类包括：TPreDef(预定义类)，TStrTool(字符串工具类)，TPathTool(路径工具类)
TFile(文件类)， TFileTool(文件工具类)， TDir(目录类)， TDirTool(目录工具类)，TTime(时间类)，
TTimeTool(时间工具类)， TLog(日志文件类)， TXMLTool(xml解析工具类)，TINI(ini文件类)
    下面是每个类的简单介绍

    1、TPreDef
    这是库中的预定义类，是所有库类的基础类，里面仿照errno，定义了一系列错误码和相应的描述信息。在开发过程中，各种错误层出不穷，很多时候想找出错误只能通过调试，这样过于繁琐。使用errno也有他的局限性。而使用了该类之后，有可能发生错误的方法都将以错误码返回，在使用过程中可以依据错误码定位错误，或者以全局函数GetDescrption获取错误信息，方便进行debug操作和日志输出。
    
    2、TStrTool
    这是库中的字符串工具类，里面实现了很多常用的字符串操作，具体可看文件内容。这里要说明一点的是，因为c++中有两种字符串，Cstyle字符串—char*， C++字符串—string。Cstyle字符串的使用往往比较麻烦和危险，而且不支持可变长度，但是开发过程中又难免要使用它，所以我在类中重载了很多函数，支持了对string和char*的混合使用，但结果都是以string返回，避免不必要的麻烦。

    3、TPathTool
    这是库中的路径工具类，实现了对路径的常见操作。所谓路径即为某个文件或者目录的相对路径或者绝对路径，该类对这些路径提供统一的操作，不区分文件与目录。

    4、TFile与TFileTool
    这两个是库中的文件与文件工具类，支持了对文件所有常见操作，包括TFile提供的读写文件内容，TFileTool提供的创建、删除、复制等操作。

    5、TDir与TDirTool
    这两个是库中的目录与目录工具类，支持了对目录所有常见操作，包括TDir提供的目录列表操作，TFileTool提供的创建、删除、复制等操作。

    5、TTime与TTimeTool
    这两个是库中的时间与时间工具类，支持了对时间所有常见操作，包括TTime提供的时间格式化操作，TTimeTool提供的提供系统时间，定时器等操作。
    
    6、TLog
    这个是库中的日志类，继承了文件类，覆盖了部分方法，添加了锁的机制，方便去输出日志文件
    
    7、TXMLTool
    这个是库中的xml解析工具类，xml简单方便，可以用于网络上传输数据，鉴于网上已经有了功能更强大的解析xml的库，该库被设计来仅用于解析与配置简单的xml
    
    8、TINI
    这个是库中的ini文件类，大部分c++开发需要用到ini文件解析初始化数据，该库可用来解析这些文件，导入和导出数据。
